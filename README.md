# Network & Web tools
Docker image with useful tools for Networking and Web Debug:

* apache2-utils
* ca-certificates
* curl
* dnsutils
* git
* netcat
* nginx
* nmap
* pv
* openssl
* traceroute
* tcpdump
* vim
* wget

### Get into the container

* `docker run -ti --entrypoint /bin/sh vincoll/webtools`

Or

* `docker exec -ti <container> bash `
